# Hangman Game

A browser-based Hangman game built using HTML, CSS (Bulma framework), and JavaScript. 

## Features

- Draws the Hangman figure using SVG and updates the drawing based on incorrect guesses
- Implements core game logic including:
  - Letter validation
  - Score tracking
  - Win/loss conditions
  - Game progression
- Generates a dynamic on-screen keyboard using JavaScript DOM manipulation
- Provides visual feedback for correct and incorrect letter guesses
- Includes "New Game" and "Skip Word" functionality
- Fetches random words from an external data source and stores them for gameplay
- Updates the UI dynamically based on player actions

## Technologies Used

- HTML
- CSS
- Bulma CSS Framework
- JavaScript
- SVG
- DOM Manipulation
